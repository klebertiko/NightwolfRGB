Quick Start Guide

Get your lighting synchronized in under 5 minutes. No complex configuration required.

1

Download OpenRGB

Install OpenRGB and start the SDK Server on port 6742. This is the core driver Nightwolf communicates with.

2

Init Workspace

Clone the repository and install dependencies using npm install in both backend/ and frontend/ directories.

3

Launch & Sync

Run npm run dev from the root. Open the dashboard, click "Cleanup" to remove conflicts, and enjoy universal control!
